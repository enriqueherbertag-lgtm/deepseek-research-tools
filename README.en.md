Thousands of scientific papers are published every month. No one can read them all. Identifying key contributions, methodologies, and limitations takes hours or days.

DeepSeek Research Tools solves that problem.

## What it does

A set of Python tools that use the DeepSeek API to accelerate scientific research.

**Paper Analyzer**: automatically extracts key contributions, methodologies, and limitations from academic papers.
**NeuroFarmac Integration**: applied examples for the NeuroFarmac project.
**Research Workflow Automation**: literature review and hypothesis generation automation (in development).

## Who it is for

- Researchers needing to process large volumes of literature.
- Academics looking to accelerate their workflow.
- Developers working with DeepSeek AI.
